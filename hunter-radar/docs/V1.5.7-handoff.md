## 五、C-6 静态分析 self_test_harness

### 5.1 核心特性

| 特性 | 说明 |
|------|------|
| 自动发现 | `DEFAULT_SCRIPTS` 25 候选 m*t*_test_*.py |
| 聚合型自动跳过 | `AGGREGATOR_PATTERNS` 4 项(m7t1/m8t1/m9t1/self_test) |
| pattern filter | `--pattern m11*` glob 通配符,自动归一化加 `*` 后缀 |
| skip-self | 排除 m15t2 自身测试,避免嵌套死锁 |
| dry-run | 只列出要跑的脚本,不真跑 |
| JSON 报告 | `--report-json path` 落地完整结构化报告 |

### 5.2 工具调用

```bash
# 默认全量
py -m scripts.self_test_harness

# 只跑 m11
py -m scripts.self_test_harness --pattern m11*

# 跑 m9t3 + m12t1
py -m scripts.self_test_harness --pattern m9t3 m12t1

# dry-run
py -m scripts.self_test_harness --dry-run

# JSON 报告
py -m scripts.self_test_harness --report-json docs/harness-report.json
```

### 5.3 嵌套死锁防御(三重)

1. **聚合型自动跳过**(`AGGREGATOR_PATTERNS`):避免 harness 调 m8t1 → m8t1 调 m9t1 → m9t1 调 harness 死循环
2. **`--skip-self` 选项**:排除 m15t2 自身测试,避免 m8t1 调 m15t2 → m15t2 调 m8t1 嵌套
3. **freeze_check §9 调 m8t1**:在 m15t1 测试中只检查函数签名不真跑,避免 m15t1 调 freeze_check → freeze_check 调 m8t1 → m8t1 调 m15t1 嵌套
